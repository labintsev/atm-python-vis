# 📻 Radio Schedule Flask Application

Интерактивное веб-приложение для визуализации расписания радиостанций ATM.

## 🎯 Функциональность

- **Главная страница**: Список всех доступных радиостанций из базы данных
- **Выбор даты**: Введите дату начала расписания (по умолчанию 2026-04-01)
- **Интерактивная визуализация**: Plotly график расписания на 7 дней вперёд
- **Детальная информация**: При наведении на ролик видна информация о нём (название, длительность, заказчик, ответственный)

## 📋 Требования

- Python 3.8+
- MS SQL Server доступен по адресу `localhost:1435`


## 🚀 Быстрый старт

### Windows (простой способ)

Просто запустите файл `run.bat`:
```bash
run.bat
```

### Windows (PowerShell)

```powershell
# Активировать виртуальное окружение
.\.venv\Scripts\Activate.ps1

# Установить зависимости (если нужно)
python -m pip install -r requirements.txt

# Запустить приложение
python app.py
```

### Linux/Mac

```bash
# Активировать виртуальное окружение
source .venv/bin/activate

# Установить зависимости (если нужно)
pip install -r requirements.txt

# Запустить приложение
python app.py
```

## 🌐 Использование

1. Откройте браузер и перейдите на `http://localhost:5005`
2. Выберите дату начала расписания в поле "📅 Дата начала расписания"
3. Кликните на одну из радиостанций
4. На новой странице откроется интерактивный Plotly график расписания на 7 дней

## 📊 Взаимодействие с графиком

- **Наведение мышки**: Наведите на точку для просмотра деталей ролика
- **Масштабирование**: Используйте скролл мышки для изменения масштаба
- **Панинг**: Зажмите левую кнопку мышки и перемещайте для сдвига
- **Сброс масштаба**: Двойной клик на график вернёт стандартный масштаб
- **Цветовое кодирование**: Каждый цвет представляет ответственного за ролик

## 🐛 Устранение неполадок

### "Ошибка: соединение с базой данных не установлено"
- Убедитесь что MS SQL Server доступен на `localhost:1435`
- Проверьте переменные окружения в файле `.env`
- Проверьте что пользователь `ReadOnlyUser` имеет права доступа к базе

### "Нет данных для выбранной радиостанции"
- Выберите другую дату
- Проверьте что на эту дату есть запланированные трансляции
- Убедитесь что радиостанция активна в этот период

### Flask приложение не запускается
- Убедитесь что виртуальное окружение активировано
- Установите зависимости: `pip install -r requirements.txt`
- Проверьте что на порту 5000 ничего не запущено

## 📝 Примечания

- Данные берутся из таблиц `Scheds_V` и `Orders_V` в базе данных
- Каждая радиостанция показывает расписание ровно на 7 дней от выбранной даты
- Применяется кодирование по цветам для разных ответственных
- Приложение работает в режиме отладки (debug=True) - для продакшена отключить

## 🔐 Безопасность

- `.env` файл содержит чувствительные данные - не коммитить в репозиторий!
- В продакшене использовать переменные окружения системы вместо `.env`
- Изменить `SECRET_KEY` в `app.py` на случайное значение для продакшена
